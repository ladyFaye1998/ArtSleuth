Methodology

The Morellian Tradition

In the 1870s, Giovanni Morelli proposed that attribution should focus on incidental details — the shape of fingernails, the curl of earlobes, the rendering of drapery folds. These peripheral passages, executed habitually and without deliberation, reveal the artist’s hand more reliably than any consciously composed focal passage.

ArtSleuth formalises the Morellian intuition as a feature-extraction problem: the “incidental details” correspond to the low-level textural and structural features that self-supervised vision transformers encode.

Dual Backbone

DINOv2 & CLIP

DINOv2 learns visual representations without labelled data via self-distillation. The resulting feature space encodes texture, directionality, and granularity — the physical surface qualities that connoisseurs evaluate.

CLIP encodes images and text in a shared embedding space. Critical for style classification, where categories like “Baroque” are culturally constructed labels — not purely visual properties.

Pipeline

Preprocessing & Patch Extraction

Standard vision-transformer preprocessing: resize, centre-crop, and ImageNet normalisation. Optional art-specific corrections: varnish correction, craquelure suppression, and canvas texture normalisation.

The painting is divided into local patches using Grid, Salient, or Adaptive strategies (default: grid + saliency-weighted oversampling).

Pipeline

Brushstroke & Style

For each patch, the 2×2 structure tensor is computed. Eigenvalues yield orientation, coherence, and energy. DINOv2 patch embeddings are clustered (k-means) to reveal homogeneous brushwork regions.

Coherence = (λ1 − λ2) / (λ1 + λ2)  Energy = √(λ1 + λ2)

Full-image CLIP embeddings are projected through three independent linear heads (period, school, genre), producing calibrated probability distributions.

Pipeline

Attribution & Forgery Detection

The fused embedding (CLIP style + mean DINOv2 patch) is compared against a reference gallery using temperature-scaled cosine similarity. Confidence intervals via normal approximation.

A one-class anomaly detector evaluates Mahalanobis distance from the reference distribution, normalised to a 0–1 anomaly score via sigmoid transform.

DM(x) = √( (x − μ)T Σ−1 (x − μ) )
Pipeline

Fusion, Temporal Drift & Workshop

Style-guided cross-attention lets CLIP embeddings (queries) attend over DINOv2 patch tokens (keys/values). 8 heads, 512 internal dim, learned temperature. Used during end-to-end training.

GP with RBF + White noise kernel models temporal style drift. Bayesian Gaussian mixture with Dirichlet process prior infers the number of hands in workshop paintings.

Caveats

Limitations

  • Training data bias: Pre-trained backbones encode biases from predominantly Western photographic training data.
  • Probabilistic, not definitive: All outputs are statistical estimates informing expert judgement.
  • Reference corpus dependency: Attribution and forgery detection quality scales with reference gallery size.

# Install
pip install artsleuth

# Quick analysis
artsleuth analyse painting.jpg

# With web UI
pip install artsleuth[web,benchmarks]
artsleuth demo

# Style classification only
artsleuth style painting.jpg

# Compare two works
artsleuth compare a.jpg b.jpg

Python API

# Full pipeline
from artsleuth import analyse

result = analyse("painting.jpg")
print(result.summary())

# Style classification
from artsleuth import StyleClassifier
clf = StyleClassifier()
report = clf.classify(image)

# Forgery screening
from artsleuth import ForgeryDetector
det = ForgeryDetector()
result = det.detect(image,
