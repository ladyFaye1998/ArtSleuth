ArtSleuth

Where connoisseurship meets computation

A computational art-analysis framework that formalises what connoisseurs have done for centuries — examining the physical evidence a painter leaves on a canvas — using vision transformers, Bayesian methods, and cross-attention feature fusion.

Capabilities

Six analytical axes, each grounded in both art-historical practice and modern computer vision

🖌️

Brushstroke Forensics

Structure-tensor analysis and coherence clustering reveal the characteristic facture of individual painters — the rhythmic signatures embedded in paint application.

Structure Tensor DINOv2 K-Means
🎨

Style Classification

Multi-axis classification along period, school, and technique using CLIP zero-shot embeddings — mapping a work's position in the broader landscape of art history.

CLIP Zero-Shot Multi-Axis
🔍

Artist Attribution

Cosine-similarity ranking against a reference gallery with confidence intervals, cross-validated by temporal plausibility scoring via Gaussian process models.

Cosine Similarity Bayesian CI
🛡️

Forgery Detection

Anomaly-based screening with z-score indicators across multiple feature dimensions, hardened against adversarial forgery simulations including age-craquelure and patina.

Anomaly Detection Adversarial Robust
👥

Workshop Decomposition

Dirichlet Process Gaussian Mixture Models automatically infer how many distinct hands contributed to a painting — a question central to Old Master attribution.

DPGMM Bayesian Non-parametric
⏳

Temporal Dating

Gaussian process regression over dated reference embeddings models an artist's stylistic evolution — estimating when a work was created and whether it fits the chronological arc.

Gaussian Process Style Drift
✦

What's Novel

Four contributions that, to our knowledge, have not been combined in an open-source art-analysis tool

1

Cross-Attention Backbone Fusion

CLIP queries attend to DINOv2 keys/values, letting semantic style understanding guide patch-level structural analysis.

2

Temporal Style Drift

GP regression over dated embeddings models an artist's evolving style — detecting anachronisms and estimating creation date.

3

Workshop Decomposition

Bayesian non-parametric mixture models infer the number of hands in a painting without a priori hand count.

4

Adversarial Forgery Robustness

Systematic simulation of historical forgery techniques — varnish darkening, craquelure, patina — to stress-test the detector.

Getting Started

From pip install to first analysis in under a minute

# Install core package
pip install artsleuth

# Quick analysis
artsleuth analyze painting.jpg

# With all optional features
pip install artsleuth[web,benchmarks]

# Launch the interactive web demo
artsleuth demo

# Run benchmarks on WikiArt
artsleuth benchmark --device cuda
# Python API
from artsleuth import analyze

result = analyze("painting.jpg")
print(result.summary())
